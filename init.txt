### Game Phases (Summary)
1. RUNNING - Mash Right Arrow to build speed and reach cart (150m base)
2. CART_RIDE - Auto-enter cart at cart position, ride to ramp (500m base), can use NOS (SPACE), hit boost pads
3. JUMPING - Cart travels up ramp, launches off
4. FLYING - Jump off ramp, steer with Left/Right arrows, spin for bonus
5. LANDING - Land to get distance score (only counts after ramp end)

### Distance System
- 1 meter = 10 pixels
- Hero starts at 0m
- Cart appears at 150m (1500px) - base, scales with trackExtender upgrade
- Ramp starts at 500m (5000px) - base, scales with trackExtender upgrade
- Distance display should feel human-scale (not too fast)

### Controls
- **Right Arrow (MASH):** Build speed while running
- **SPACE (HOLD):** NOS boost during cart ride
- **Left/Right Arrow (HOLD):** Spin the cart during flight
- **U:** Open upgrade shop
- **M:** Toggle sound
- **P:** Pause game
- **R:** Reset game

### Key Features (v1.5)
- Height warning display during flight (center screen)
- Distance milestone popups (500m, 450m, 400m...)
- NOS boost system with boost pads on track
- Upgrade shop with 8 categories (20 tiers each):
  - Shoes (max speed)
  - Ramp Extender (ride)
  - Distance Amplifier (distance)
  - NOS Power (nosPower)
  - NOS Tank (nosTank)
  - Turbo Pads (boostPads)
  - Boost Duration (boostDuration)
  - Track Extender (trackExtender) - scales all phases
- Profile save system (3 slots) with file import/export
- Spin tracking with gold multiplier
- Objectives system
- Technical diagnostics panel
- Console system with cheat menu
- Custom NOS colors (unlocked at high levels)
- Extended track system with boost pads

### Track Extender System
- Each level adds 5% to run and cart phase distances
- Boost pads increase by 1 every 4 levels
- Base: 150m to cart, 500m to ramp
- Max (level 20): ~285m to cart, ~950m to ramp
